# Release Notes

## v1.0.7-beta

### Highlights
- Native compiler generates assembly for both x86_64 and ARM64 hosts.
- Compiler/transpiler build matrix covers Linux, Windows, and macOS on x86_64 and ARM64.
- Smoke tests validate basic compiler and native compiler output generation.

### Components
- **sub**: Compiler/transpiler driver.
- **subc**: Native compiler driver.

### Supported Host Targets
- Linux: x86_64, ARM64
- Windows: x86_64, ARM64
- macOS: x86_64, ARM64
